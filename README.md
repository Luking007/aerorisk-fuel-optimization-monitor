# ⛽ AeroRisk — Fuel Optimization Monitor

**AI-powered fuel efficiency anomaly detection for Nigerian domestic aviation**

Built by [Oyewo Lukman Segun](https://linkedin.com/in/oyewo-lukman-segun)

## 🔍 Problem
Fuel represents 30-40% of airline operating costs. Nigerian carriers 
lack systematic anomaly detection to catch inefficient fuel burn 
before it compounds into significant financial loss.

## 💡 Solution
Automated system comparing planned vs. actual fuel burn across 
600 flights, flagging anomalies by severity and root cause, with 
automated weekly cost-impact alerts.

## 📊 Key Findings
- ₦98.9M total excess fuel cost identified (6-month period)
- ₦197.8M annualized projection
- 37 severe anomalies = ₦41.7M impact alone
- Worst aircraft (5N-BWF): 5.06% avg fuel variance
- Worst airline (Overland Airways): 4.55% avg variance
- Top causes: Weather diversion, non-optimal altitude, excess taxi time

## 🛠️ Tech Stack
Python (pandas, numpy) · Statistical Anomaly Detection · 
Power BI · n8n · Google Sheets · Gmail Automation

## 📧 Automation
Weekly automated fuel efficiency alert delivered every Monday 
7:30AM — zero manual intervention.
